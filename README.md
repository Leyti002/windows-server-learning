# 🖥️ Windows Server Learning Journey

Ce dépôt documente mon apprentissage de Windows Server à travers le cours OpenClassrooms :
👉 "Prenez en main Windows Server"

## 🎯 Objectifs

- Comprendre l’architecture de Windows Server
- Installer et configurer un serveur Windows
- Maîtriser Active Directory
- Gérer les utilisateurs et les groupes
- Administrer les ressources réseau
- Développer des compétences en administration système
- Se préparer à une carrière en cybersécurité (pentesting / admin système)
